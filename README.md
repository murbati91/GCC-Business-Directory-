# GCC Business Directory 🌍 

A comprehensive business directory platform for the Gulf Cooperation Council (GCC) region.

## 🏳️ Supported Countries

| Flag | Country | Code |
|------|---------|------|
| 🇧🇭 | Bahrain | BH |
| 🇰🇼 | Kuwait | KW |
| 🇴🇲 | Oman | OM |
| 🇶🇦 | Qatar | QA |

| 🇸🇦 | Saudi Arabia | SA |
| 🇦🇪 | United Arab Emirates | AE |

## ✨ Features

- **🔍 Search & Filter** - Find businesses by name, category, location, or keywords
- **🏢 Business Profiles** - Detailed company information with contact details
- **📱 Mobile Responsive** - Optimized for all devices and screen sizes
- **🚀 SEO Optimized** - Built for search engine visibility

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/murbati91/GCC-Business-Directory.git

# Navigate to directory
cd GCC-Business-Directory

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## 📦 Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
